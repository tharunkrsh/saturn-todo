# SATURN — Task Machine

An intelligent offline task manager with smart scheduling and streak-based rewards.

## 🔗 Live Demo
https://saturn-todo.netlify.app

## Background
Original prototype made in Python/Jupyter using ipywidgets and JSON file storage before 
being rebuilt as a full web app. See `saturn_prototype.ipynb`.

## Features
- Smart task input with urgency (0-10) and task size (1-5)
- Auto-generated 7-day schedule based on your free slots
- Fixed appointment and recurring slot management
- Streak tracker with 5 unlockable reward themes (Bronze, Silver, Gold, Diamond)
- 3D animated logo built with Three.js
- Completion animations and progress graphs
- Fully offline, all data persists in localStorage

## Tech Stack
JavaScript · HTML/CSS · Three.js · Tailwind CSS · localStorage API
